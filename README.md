# graduate-manage-system
毕业生就业管理系统（持续更新中...）

### 主要信息
- 后端：Spring boot + Mybatis + Maven + MySQL 
- 前端：Bootstrap + Thymeleaf
- 页面模板来自：模板之家

### 主要功能：
1. 学生
   1. 登录注册
   2. 管理个人信息
2. 公司
   1. 登录注册
   2. 发布宣讲会信息
   3. 发布招聘信息
   4. 管理个人信息
   5. 管理自己发布的信息
3. 管理员
   1. 登录
   2. 管理所有人员信息
   3. 管理所有宣讲会信息
   4. 管理所有招聘信息
   5. 管理新闻、公告等信息
